# React Cinema Booking App

Small booking cinema app.

## Запуск проекта

Для работы с проектом вам понадобится [Node.js](https://nodejs.org/).

### 1. Установка зависимостей

Перед первым запуском установите все необходимые пакеты:

```bash
npm install
```

### 2. Запуск с моками (Рекомендуется для разработки)

Чтобы запустить приложение с использованием Mock Service Worker (MSW) для имитации бэкенда:

```bash
npm run msw
```

- Приложение будет доступно по адресу: [http://localhost:3000](http://localhost:3000)
- API запросы будут перехватываться MSW.

### 3. Обычный запуск

Для запуска приложения без моков (ожидается реальный бэкенд на порту 3022 или другом настроенном):

```bash
npm start
```

- Приложение будет доступно по адресу: [http://localhost:3022](http://localhost:3022) (по умолчанию)

### 4. Тестирование

Для запуска тестов:

```bash
npm test
```

### 5. Линтинг

Для проверки кода линтером:

```bash
npm run lint
```
